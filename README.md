# Tahj's Portfolio 🖥️

My personal portfolio site — designed and built from scratch to showcase my front-end projects and skills as a self-taught developer.

## Live Demo

https://then25.github.io/tahj-portfolio/

## About

This is my personal portfolio site, built without any frameworks or templates. In the spirit of full transparency — because honesty is a core value I hold as a Christian — this site was built with the assistance of Claude (Anthropic's AI). I am capable of building independently, and I used AI intentionally to increase my efficiency and bring my ideas to life faster. The creative direction, design decisions, and vision behind the site are entirely my own. I also believe that knowing how to work effectively with AI tools is a valuable skill in today's development landscape, and I'm confident in my ability to do so with purpose and integrity.

## Features

- Responsive layout that works on mobile, tablet, and desktop
- Fixed navigation bar with a working hamburger menu on mobile
- About section with a two-column layout, profile photo, and skill pills
- Projects section showcasing my work with live and GitHub links
- Skills section organized by category
- Contact section with email, GitHub, and LinkedIn links
- Fade-up scroll animations using IntersectionObserver
- CSS custom properties for consistent theming
- Google Fonts — DM Serif Display & DM Mono

## Built With

- HTML5
- CSS3 (Flexbox, Grid, Custom Properties)
- Vanilla JavaScript
